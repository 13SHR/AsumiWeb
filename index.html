Bienvenue sur le site décrivant ce studio

Bienvenue sur le site officiel de Asumi Studio ! Nous sommes un surdio de développement de jeux vidéos, dans lequel tous les membres, développeurs graphistes ou même compositeurs, s'unissent pour créer à partir de 0 un jeu entier.
Ce sudio a été créé à partir de 3 personnes ayant les mêmes ambitions : créer des jeux pendant notre temps libre.

Pour en savoir plus, vous pouvez naviguer sur ce site qui décrit notre passion et l'histoire du projet.
